3 複製資料 → 點書籤填入

Proactis 每 10 筆換一頁，所以資料會自動分頁。請依序操作：

  1. 在 Proactis 依上方順序新增好所有 Item Code
  2. 點下方對應頁的「複製」按鈕
  3. 到 Proactis 頁面，點書籤列上的「📋 填入 Proactis」
  4. 在彈出的視窗中貼上（Ctrl+V），點「填入全部」
  5. 如果有下一頁，在 Proactis 點「▶」翻到下一頁，再重複步驟 2-4
